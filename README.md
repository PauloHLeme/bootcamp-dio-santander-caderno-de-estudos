# Bootcamp Santander 2025 - Ciência de Dados com Python

## Objetivo Deste Caderno

Este é um projeto criado por mim, [Paulo Henrique Leme](https://github.com/PauloHLeme/), para servir primariamente como caderno de estudos do Bootcamp de Ciência de Dados da DIO em parceria com o Santander.

Decidi tornar esse projeto público para que outras pessoas participando de iniciativas parecidas tenham como inspiração o projeto e possam aprender com meus resumos, me ajudar adicionando informações que não coloquei e também quem sabe criarem seus próprios projetos.
