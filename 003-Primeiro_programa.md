# Nosso primeiro código em Python

## A Receita

O computador é um tremendo especialista quando o assunto é seguir instruções.

Quando criamos um código, quando programamos, estamos criando uma sequência de rotinas, uma lista de passos para que o computador siga e gere o resultado esperado.

Fazer um código é bem parecido com fazer uma receita. Nós damos o passo a passo da receita, junto com os ingredientes necessários, e o computador só segue essa receita, para nos entregar o bolo fresquinho sem que a gente precise colocar a mão na massa.

## Criando um arquivo

Para criarmos um arquivo que o computador saiba que é feito em Python, nos colocamos a extensão `.py` no final do nome do arquivo. Dessa forma, fica identificado o que o computador deve utilizar para ler a receita que criamos

### O código

Com o VSCode aberto, vamos criar um novo arquivo e chamar de `primeiro-codigo.py`. Agora, dentro do código, podemos criar um simples comando que vai enviar uma mensagem na tela.


``
print("Hello, World!")
``

Simples assim! Agora, no canto superior direito do VSCode temos uma *setinha* que serve para rodar o código, e vamos ver ele funcionando no terminal e nos dando o olá.

**Parabéns!** Já criamos nosso primeiro código em Python.
