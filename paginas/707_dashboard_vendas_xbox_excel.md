# Criando um Dashboard de Vendas do Xbox com Excel

## Método ABCDE

### A (Assets)

São todos os recursos de apoio visual do Dashboard. Isso vale para imagens, videos, gifs, paletas de cores...

### B (Bases)

São os dados que serão utilizados para gerar a dashboard principal ou outras medidas

### C (Cálculos)

São as fórmulas e cálculos utilizados para chegar na informação, o meio de campo que processa os dados para gerar as informações

### D (Dashboard)

É o painel visual, onde as informações serão exibidas para o seu cliente

### E (Extras)

Este elemento pode ou não existir, que são elementos adicionais, como documentações, pdf's, links...

## Especificidades da Nossa DB

### Empilhamento de Serviços

Uma categoria de alguns negócios é o Upsell, ou o Empilhamento de Serviços.

Esse modelo de negócio visa agregar a um primeiro produto a possibilidade de acréscimo de um segundo, terceiro ou assim por diante mediante um pagamento extra. Quando você compra um lanche e por mais 5 reais pode levar também batata e refri, você adquiriu um Upsell. Quando você manda lavar o carro e por mais 10 reais leva o polimento, também está levando um serviço extra, empilhado.

## Dados vs Informações

Um dado é um registro de um valor, que pode ser quantitativo ou qualitativo, mas que sozinho, representa apenas os elementos que o compoem, mas não responde a uma pergunta de negócio.

Já a informação é o ato de **uma pergunta de negócio sendo respondida através de alguma análise de dado específica**.

Essas análises são dadas através de Cálculos.

Dados de vendas com empilhamento de serviço são uma coisa, agora, responder a _"Quanto faturamos com a venda dos três serviços dentro do plano Anual?"_ é responder com a análise de dados a uma pergunta de negócio, ou seja, é uma **Informação**.

## Data View

Todas as informações podem ser exibidas de maneira visual, seja utilizando textos, mas principalmente utilizando cores, ícones, elementos gráficos e outros elementos que saltam aos olhos.

Em Dashboards, as informações não estão colocadas de forma visual para serem bonitinhas, pelo menos não apenas. Elas servem para trazer as informações de forma visual clara, que faz com que o usuário do Dashboard seja de fácil interpretação para qualquer cliente.
