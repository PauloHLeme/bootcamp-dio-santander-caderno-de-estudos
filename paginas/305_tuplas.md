# Tuplas

Tuplas são muito parecidas com as listas, porém elas não podem ser alteradas.

## Criando tuplas

Podemos criar tuplas colocando os itens dentro de parênteses, separados por vírgulas.

```python
veiculo = ("GM","Vectra",2010,"preto","automático",)

paises = ("Brasil","China",)
```

### _Boas Práticas_
Tuplas são declaradas com parênteses, que são os mesmos símbolos para realizar ordem de precedência aritméticas.

> Por isso, é uma boa prática **adicionarmos uma vírgula após o último elemento da tuplo**, removendo assim a possibilidade de que o Python leia o código de forma errônea.
