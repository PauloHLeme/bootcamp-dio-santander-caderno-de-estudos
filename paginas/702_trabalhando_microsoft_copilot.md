# Trabalhando com o Microsoft Copilot

## Conhecendo o Copilot

O Microsoft Copilot é a Ferramenta de IA da Microsoft onde nós podemos conversar com ela de forma natural. É a IA oficial da Microsoft, que vem como uma alternativa ao ChatGPT da OpenAI e o Gemini do Google.

Ele consegue trabalhar com conversação em diversos idiomas, ler documentos e ainda trabalhar com criação de imagens.

## Cuidados com os Resultados

As IAs estão cada vez mais inteligentes e mais conectadas com a internet para a consulta rápida dos dados atualizados, mas mesmo assim, delays e erros podem acontecer. Aliás, não é algo incomum de acontecer e nem é esperado que ela seja infalível.

Por isso, podemos utilizar a IA como uma excelente assistente, mas não como uma fonte segura e completamente confiável de dados. Desta forma, é bom dizer:

> **NÃO USARÁS IA PARA FABRICAR DADOS**

Ao invés disso, podemos utilizar ela para nos auxiliar a **Analisar dados já existentes**.

## Testes de Realidade

Testes de realidade são aqueles testes executados para sabermos se a IA está se comportando de forma esperada, retornando dados corretos e não tendo o que chamamos de alucinações.

Para isso, podemos perguntar para ela algo sobre a qual já sabemos a resposta ou algo que seja de fácil pesquisa em uma fonte confiável. Desta forma, se o resultado for o esperado, temos um nível maior de confiabilidade na utilização desta ferramenta.

## Dados sensíveis

Importante dizer que, mesmo em ambientes controlados e internos, é sempre bom manter dados sensíveis como senhas, chaves de API, dados pessoais de usuários e segredos de empresa fora das bases de conhecimento da IA, para que não exista o risco de que tais informações acabem sendo vazadas dentro desta base ou até mesmo por compartilhamento das informações da IA com outros usuários.
