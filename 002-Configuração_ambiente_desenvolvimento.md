# Configuração de Ambiente de Desenvolvimento

Para seguir o curso da melhor forma possível, vamos configurar a máquina de acordo com as aulas.

## Instalando o Python

Para usuários *Linux* e *MacOS* o Python já costuma vir instalado por padrão.

Para usuários Windows, você pode instalar o Python através [desse link](https://www.python.org/downloads/ "Download do Python").

## Instalando a IDE

IDE é a interface de desenvolvimento que vamos utilizar. É como um editor de texto, mas que é especialmente feito para se trabalhar com códigos.

Vamos utilizar o **Visual Studio Code**. Você pode fazer o download [clicando aqui](https://code.visualstudio.com/download "Download do Visual Studio Code").

### Instalando extensões no Visual Studio Code

O Visual Studio Code (VSCode) é uma IDE modular, ou seja, permite o suporte a instalação de vários módulos que servem para expandir a capacidade de serviço da IDE de acordo com a necessidade de cada desenvolvedor.

Vamos instalar uma extensão que aplica um suporte mais completo ao nosso código Python através [desse link](https://marketplace.visualstudio.com/items?itemName=ms-python.python "Extensão Python").

Alternativamente, podemos clicar na aba de extensões do VSCode e pesquisar por **Python** e instalar a extensão que aparecer com o  mesmo nome.

Vamos instalar também a extensão [autoDocstring](https://marketplace.visualstudio.com/items?itemName=njpwerner.autodocstring "Extensão Auto Docstring"), que vai nos auxiliar a gerar a documentação do código.
